# AutoMux
本工具是为了懒人更便捷的配置Termux所写.

## 截图
<img src="Module/AutoMux.png">

### 依赖
• Python 3.x

##### 手动安装方法:

**如果觉得Github速度慢，可以使用国内码云链接**

https://gitee.com/whitedays2007/automux

git clone https://github.com/whitedays2007/automux $PREFIX/share/

cd $PREFIX/share/AutoMux

chmod +x Module/*

直接在环境中启动 automux
